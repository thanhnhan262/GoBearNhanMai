>>>Prerequisites:
- Java 8 is installed in executing machine
- IntelliJ is installed in executing machine
- Chrome 81 is installed in executing machine

>>>How to run test:
- Open the project in IntelliJ, wait for dependencies downloaded completely
- For executing all test suite: Right-click on src/test/testsuite/regression.xml, select Run.
- For executing a particular test: Go to testing classes (src/test/java/strategy/regression), right-click on a test method end select Run.
